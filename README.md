# CardioCare AI
Sistem prediksi risiko penyakit kardiovaskular berbasis *deep learning* (TensorFlow) untuk Capstone Project Coding Camp 2026.

## 📋 Deskripsi
CardioCare AI adalah alat bantu skrining awal yang memprediksi risiko kardiovaskular dengan *Recall* 85.94% menggunakan *Threshold Medis* 0.40.
